# MySQL DB dumper

This image is a direct port of [PostgreSQL DB dumper](../postgres-dumper/readme.md)
with very few changes.

## Variables

- `MYSQL_USER`: `root`
- `MYSQL_PASSWORD`: `1234`
- `MYSQL_HOST`: `localhost`
- `MYSQL_PORT`: `3306`
- [`BOTO_PATH`][boto-path]: empty
- `GS_PATH`: empty
- `GOOGLE_APPLICATION_CREDENTIALS`: empty
